# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/)
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Command palette (Cmd/Ctrl+K) with invoice search, page navigation, and action commands.
- Changelog modal with version tracking and formatted release notes.
- Footer with version display and changelog link.

## [0.1.0] - 2026-05-18

### Added
- Initial public scaffold of the Kora-Frontend repository.
- Invoice marketplace with filtering, sorting, and search.
- SME dashboard for managing tokenized invoices.
- Investor dashboard with position tracking and yield projections.
- Wallet integration via Stellar Wallets Kit (Freighter, xBull, LOBSTR, Albedo).
- Invoice creation flow with IPFS document upload.
- On-chain invoice funding and repayment via Soroban smart contracts.
- Analytics page with charts and portfolio metrics.
- Transaction history with full lifecycle tracking.
- PWA support with offline page and install prompt.
- Onboarding tour for new users.
- Dark/light/system theme support.
- Address book for saved Stellar addresses.
